# sushi-css-html

## Sample CSS and HTML only UI page

This is a sample of a UI page using only CSS and HTML based on JSMasteryPro course.  I have updated the code to have CSS default values for fonts.
Created using `npm create vite@latest`

## How to run

After pulling the code from GitHub,

Type the following command in the terminal.

  `npm install`


  `npm run dev`


  Navigate to http://localhost:5173
